# Lux Wallet: HD Wallet with Multi-Chain and Ledger Support

Managing assets across Lux's three chains (X-Chain, P-Chain, C-Chain) requires a wallet that understands the multi-chain architecture natively. Today we are launching Lux Wallet: an HD wallet built specifically for the Lux ecosystem.

## Features

### Hierarchical Deterministic (HD)

BIP-32/BIP-39 compliant HD wallet:

- Single mnemonic phrase for all chains
- Deterministic key derivation — restore from seed on any device
- Multiple accounts from a single seed

### Multi-Chain Native

One wallet, three chains:

- **X-Chain**: Asset creation, transfer, and management
- **P-Chain**: Staking, delegation, and validator operations
- **C-Chain**: EVM transactions, smart contract interaction, DeFi

Cross-chain transfers happen within the wallet — move assets between chains without external tools.

### Hardware Wallet Support

Ledger integration for cold storage:

- Transaction signing on Ledger hardware
- Full support for all three chains
- No private keys on the host device

### Staking and Delegation

Participate in network security directly from the wallet:

- Delegate to validators with one click
- Track staking rewards in real time
- Validator selection with performance metrics

## Web and Desktop

- **Web wallet**: Browser-based, no installation required
- **Desktop**: Native applications for macOS, Windows, and Linux
- **Mobile**: iOS and Android (coming soon)
