# Hanzo Auto: AI Workflow Automation

Zapier connects apps. Make (Integromat) builds workflows. Neither understands what the data means. They move information from A to B without comprehension. When the workflow needs judgment — should this email go to sales or support? Is this lead qualified? Does this order need review? — you are back to manual intervention.

Today we are launching Hanzo Auto: workflow automation with AI built into every step.

## How It Works

### Visual Builder

Drag-and-drop workflow creation:

- Trigger → Action → Condition → Action chains
- Branch logic with AI-powered decision nodes
- Loop and batch processing
- Error handling with retry policies

### AI Decision Nodes

The differentiator. AI nodes that understand context:

- **Classify**: Route data based on semantic meaning, not keyword matching
- **Extract**: Pull structured data from unstructured text (invoices, emails, documents)
- **Summarize**: Condense long documents into actionable summaries
- **Generate**: Create responses, reports, or content based on workflow context
- **Judge**: Make yes/no decisions with explanation

### 280+ Integrations

Connect to the tools your team already uses:

- **CRM**: Salesforce, HubSpot, Hanzo CRM
- **Communication**: Slack, Email, Teams, Discord
- **Development**: GitHub, GitLab, Jira, Linear
- **Commerce**: Shopify, Stripe, WooCommerce
- **Data**: PostgreSQL, MongoDB, Google Sheets, Airtable
- **AI**: OpenAI, Anthropic, Hanzo LLM Gateway

## Example: Intelligent Lead Routing

```
Trigger: New form submission
  → AI Extract: Parse company info from submission
  → AI Classify: Determine lead quality (hot/warm/cold)
  → Branch:
    Hot → Create deal in CRM + alert sales on Slack
    Warm → Add to nurture sequence in email
    Cold → Add to newsletter list
```

No rules to maintain. No keyword lists to update. The AI understands context.
