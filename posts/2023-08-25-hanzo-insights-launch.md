# Hanzo Insights: Product Analytics That Respects Privacy

Google Analytics sells your users' data to advertisers. Mixpanel charges $1,000/month for basic funnel analysis. Amplitude locks core features behind enterprise tiers. The product analytics market optimizes for the vendor, not the customer.

Today we are launching Hanzo Insights: full-featured product analytics that does not sell user data and does not require an enterprise contract.

## Features

### Event Tracking

Track user actions with flexible schema:

- Auto-capture (clicks, pageviews, form submissions)
- Custom events with arbitrary properties
- Server-side and client-side SDKs
- No sampling — every event captured and queryable

### Funnels

Understand where users drop off:

- Multi-step funnel definition
- Conversion rate by step
- Time-to-convert analysis
- Segment funnels by user properties

### Cohorts

Track user behavior over time:

- Retention curves by cohort
- Behavioral cohort definition (users who did X)
- Cohort comparison
- Lifecycle analysis (new, returning, dormant, resurrected)

### Session Replay

See what users actually do:

- Full session recording (DOM-based, not video)
- Click maps and scroll maps
- Filter replays by events, errors, or user properties
- Privacy controls (mask sensitive fields automatically)

### Feature Flags

Integrated with Hanzo Experiments:

- Track feature flag impact on product metrics
- A/B test results correlated with analytics events
- Automatic significance calculations

## Privacy

- We do not sell user data
- We do not share data with advertisers
- GDPR and CCPA compliant out of the box
- Self-hosted option for complete data sovereignty
- Cookie-less tracking mode available
