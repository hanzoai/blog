# Lux Explorer: Full Block Explorer with Contract Verification

Blockchain transparency requires tooling. Today we are launching Lux Explorer: a full-featured block explorer for the Lux network.

## Features

### Transaction Search

Find any transaction by hash, block number, or address:

- Full transaction details including input data decoding
- Internal transactions and token transfers
- Gas usage and fee breakdown
- Cross-chain transaction tracking

### Contract Verification

Verify and interact with smart contracts:

- Source code verification (Solidity, Vyper)
- Read and write contract functions from the browser
- ABI decoding for all verified contracts
- Proxy contract resolution

### Token Tracking

Complete token analytics:

- ERC-20 token balances and transfer history
- ERC-721 NFT collections and ownership
- ERC-1155 multi-token tracking
- Token holder distribution charts

### Network Statistics

Real-time network health:

- TPS and finality time charts
- Validator count and stake distribution
- Gas price trends
- Active address counts

### API

Programmatic access to all explorer data:

- REST API compatible with Etherscan API format
- Existing tools that query Etherscan work with Lux Explorer
- Rate-limited free tier, authenticated access for higher limits

## Built on Blockscout

Lux Explorer is built on Blockscout with custom modifications for Lux's multi-chain architecture. Open source and self-hostable.
