# Hanzo Blog

Official blog for Hanzo Industries.

## Structure

```
blog/
  posts/           # Blog posts in markdown
  README.md        # This file
```

## Post Format

Posts use frontmatter with the following fields:

```yaml
---
title: "Post Title"
date: "YYYY-MM-DD"
author: "Author Name"
tags: ["tag1", "tag2"]
description: "Brief description"
---
```

## Usage

Posts are organized by date in the `posts/` directory with the naming convention:
`YYYY-MM-DD-slug.md`
